## ✦ Overview

**Nexus** is a fully local, always-on voice assistant that runs quietly in your terminal, waiting to be woken up. No cloud dashboard, no subscription, no app to install on your phone — just Python, your microphone, and your own free Gemini API key.

Say **"Hey Nexus"**, ask your question out loud, and it answers — both in the terminal and out loud through your speakers. Simple local questions (time, date, quick math) are answered instantly with zero API calls. Everything else is handled by **Gemini 2.5 Flash**, with full conversation memory so follow-up questions actually make sense.

It's built to feel less like a script and more like a presence in the room.
